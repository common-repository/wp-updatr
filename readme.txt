== Description ==

= A comprehensive, easy to use Automatic Updates solution for your WordPress products =

WP Updatr provides  an easy to use platform that allows you to offer automatic updates for your WordPress products. 

By installing the WP Updatr WordPress plugin, you'll be able to link your website to our platform to automatically generate license keys for your customers.

You'll be able to specify the lifespan of a license as well as a site limit if required.

= WP Updatr Account Required =

To make use of the WP Updatr service, you'll need to sign up for an [account](https://wpupdatr.com/). Install the WP Updatr WordPress plugin to sign up for a free account without a trial.  

We've also got a [Plus Plan](https://wpupdatr.com/#pricing) that allows you to create up to 3 licensed products, as well as a [Professional Plan](https://wpupdatr.com/#pricing) that allows you to create up to 10 licensed products.

= Unlimited Updates =

Get the zip file ready, fill out the version number and enter in a changelog. Your customers are about to receive the latest version of your product within seconds.

= Unlimited Products =

We don't limit the number of products that you can create. Provide updates to as many products as you need.

= Comprehensive Reporting =

It's all in the numbers. That's why we've put together a variety of comprehensive reports that make it easy for you to keep track of the usage and performance of your product.

= Multiple Integrations =

Selling your product is easy. Getting a license generated for each customer is even easier when you're using WooCommerce, Paid Memberships Pro & Easy Digital Downloads.

== Installation ==

= Download, Install and Activate! =
1. Go to Plugins > Add New to find and install WP Updatr.
2. Or, download the latest version of the plugin, then go to Plugins > Add New and click the "Upload Plugin" button to upload your .zip file.
3. Activate the plugin.

= Complete the Initial Plugin Setup =
Go to Settings > WP Updatr in the WordPress admin to begin setup. Our [Initial Setup Tutorial](https://wpupdatr.com/documentation/getting-started-with-wp-updatr/) will guide you on linking your website to the WP Updatr Service and how to get started with creating your first licensed product.
